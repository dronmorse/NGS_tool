What does your DNA encode?

Do you have a FASTA file without a proper header? Maybe you have found an old DNA sequence on your hard drive that you have no idea what it encodes? Do not look any further! This pipeline:

  • takes a FASTA file as input or performs a test using one of the saved FASTA files. Saves the data into a local database for later usage.
  • If a file consists multiple sequences separated by headers, the user will be asked to choose only one sequence for analysis
  • The final sequence is uploaded to NCBI BLASTN server for homology assessment and a report is returned with a possible protein which is encoded by the input.
